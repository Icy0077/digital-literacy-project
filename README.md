# 📘 Digital Literacy Project

## 📌 Project Overview
This project was completed as part of the Digital Literacy course. The aim of this project is to develop essential digital skills required for students to safely and effectively participate in the digital world. As a Student Digital Ambassador, this project focuses on digital awareness, professional online presence, communication etiquette, and cyber safety.

The portfolio includes practical activities such as creating an infographic, building professional profiles, exploring coding and collaboration platforms, drafting professional emails, and researching cybercrime awareness.

---

## 🧩 Project Tasks

### ✅ Task 1 – Digital Literacy Awareness Infographic
Created a one-page infographic explaining digital literacy concepts, safe internet practices, and useful digital tools for students.

📁 Folder: `task-1-presentation/`

---

### ✅ Task 2 – Student Digital Portfolio
Set up professional profiles on platforms such as GitHub, LinkedIn, and Kaggle to build a strong digital presence and showcase learning progress.

📁 Folder: `task-2-portfolio/`

---

### ✅ Task 3 – Coding & Collaboration Platforms
Completed a beginner coding challenge and created a Digital Literacy Awareness Quiz using Google Forms.

📁 Folder: `task-3-platforms/`

---

### ✅ Task 4 – Professional Email & Etiquette
Drafted professional emails and created a social media Do’s and Don’ts checklist demonstrating proper digital communication practices.

📁 Folder: `task-4-email-etiquette/`

---

### ✅ Task 5 – Cybercrime Awareness
Prepared a case study on UPI/Online Payment Fraud and created a “Stay Safe Online” prevention checklist for students.

📁 Folder: `task-5-cybercrime/`
